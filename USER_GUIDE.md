# 🎮 TechStore 使用指南

歡迎使用 TechStore！這是一個完整的線上拍賣平台。

---

## 🚀 快速開始

### 本地運行

```bash
# 1. 安裝依賴
npm install

# 2. 啟動開發伺服器
npm run dev

# 3. 開啟瀏覽器
訪問 http://localhost:3000
```

### 第一次使用

網站會自動載入 6 個範例商品，你可以：
1. 瀏覽商品
2. 搜尋與篩選
3. 上傳自己的商品
4. 加入購物車購買
5. 與賣家聊天

---

## 📖 功能使用教學

### 1️⃣ 瀏覽商品

**首頁**（http://localhost:3000）

- **商品卡片**：顯示商品照片、標題、價格、狀態、賣家資訊
- **搜尋列**：在 Header 中間，輸入關鍵字並按 Enter
- **分類篩選**：點擊下方的分類標籤（全部、3C產品、遊戲等）
- **點擊商品**：查看完整詳情

### 2️⃣ 上傳商品

**上傳頁面**（/upload）

1. 點擊 Header 右側的橘色「上傳商品」按鈕
2. 上傳照片：
   - 方法 1：拖放照片到虛線區域
   - 方法 2：點擊「選擇檔案」按鈕
   - 最多 5 張照片
   - 點擊照片右上角的 × 可移除
3. 填寫商品資訊：
   - 商品標題（必填）
   - 商品描述（必填，詳細描述商品狀況）
   - 價格（必填，NT$）
   - 商品狀態（下拉選單：全新/二手-近全新/二手-良好/二手-普通）
   - 商品分類（下拉選單）
4. 點擊「上傳商品」按鈕
5. 自動跳轉到首頁，看到你的新商品！

### 3️⃣ 查看商品詳情

**商品詳情頁**（/product/:id）

- **圖片輪播**：點擊下方縮圖切換照片
- **商品資訊**：標題、價格、狀態、分類、描述
- **賣家資訊**：頭像、名稱、評分
- **三個行動按鈕**：
  - 💬 **聊聊**：開啟與賣家的聊天室
  - 🛒 **加入購物車**：加入購物車（會顯示提示訊息）
  - 💳 **立即購買**：直接跳轉到購物車結帳

### 4️⃣ 購物車

**購物車頁面**（/cart）

1. 點擊 Header 右側的購物車圖標
2. 查看已選商品：
   - 商品照片、標題、價格
   - 數量調整：點擊 + 或 - 按鈕
   - 移除商品：點擊「移除」按鈕
3. 查看總金額：
   - 小計：商品總價
   - 運費：NT$ 60
   - 總計：小計 + 運費
4. 點擊「前往結帳」按鈕

### 5️⃣ 結帳

**結帳頁面**（/checkout）

1. 填寫收件資訊：
   - 收件人姓名（必填）
   - 聯絡電話（必填，手機格式：0912345678）
   - 電子郵件（必填，Email 格式）
   - 收件地址（必填，完整地址）
   - 備註（選填）
2. 右側查看訂單摘要：
   - 商品列表
   - 金額明細
3. 點擊「確認訂單」
4. 看到成功提示，購物車已清空！

### 6️⃣ 聊聊系統

**聊天頁面**（/chat）

#### 開啟聊天室的方式：

**方法 1**：從商品詳情頁
1. 進入商品詳情頁
2. 點擊「聊聊」按鈕
3. 自動開啟與該商品賣家的聊天室

**方法 2**：從 Header
1. 點擊 Header 右側的聊聊圖標
2. 查看所有對話列表
3. 點擊任一對話進入聊天室

#### 聊天室使用：

**左側：對話列表**
- 顯示所有聊天室
- 賣家頭像與名稱
- 最後訊息預覽
- 時間顯示（剛剛/幾分鐘前/今天幾點/日期）
- 未讀訊息數量（紅色徽章）
- 點擊對話進入聊天室

**右側：聊天室**
- 頂部：賣家資訊
- 訊息列表：
  - 對方訊息：灰色氣泡，靠左
  - 我的訊息：橘色氣泡，靠右
  - 每則訊息顯示時間
- 底部：輸入框 + 發送按鈕
- 自動滾動到最新訊息

**手機版**：
- 對話列表與聊天室單欄切換
- 點擊對話進入聊天室
- 點擊返回按鈕回到對話列表

## 🎨 介面說明

### Header（導航列）

從左到右：
1. **Logo + 網站名稱**：點擊返回首頁
2. **搜尋列**：輸入關鍵字搜尋商品
3. **上傳商品按鈕**（橘色）：開啟上傳頁面
4. **聊聊圖標**：開啟聊天系統（有未讀數量提示）
5. **購物車圖標**：開啟購物車（有商品數量提示）
6. **用戶頭像**：目前使用者（預設：我）

## 💡 使用技巧

### 搜尋功能
- 搜尋會比對商品標題
- 搜尋結果會顯示在首頁
- 可搭配分類篩選使用
- URL 會保留搜尋參數（可分享連結）

### 購物車管理
- 商品數量可調整（最少 1 件）
- 數量設為 0 會自動移除商品
- Header 的購物車圖標顯示總商品數量
- 購物車資料會自動儲存

### 聊天系統
- 從商品詳情頁點擊「聊聊」會自動建立與該賣家的聊天室
- 對話列表按最後訊息時間排序
- 點擊對話會標記為已讀
- 所有訊息會自動儲存

### 資料儲存
- 所有資料儲存在瀏覽器 LocalStorage
- 重新整理頁面不會遺失資料
- 清除瀏覽器資料會遺失所有內容
- 首次使用會自動載入 6 個範例商品

## 🐛 常見問題

### Q: 商品上傳後看不到？
A: 重新整理頁面，或返回首頁查看。商品會顯示在列表最上方。

### Q: 購物車商品不見了？
A: 檢查是否清除了瀏覽器資料。所有資料存在 LocalStorage。

### Q: 如何刪除商品？
A: 目前版本沒有刪除功能。可清除瀏覽器 LocalStorage 重置所有資料。

### Q: 可以上傳影片嗎？
A: 目前只支援圖片（JPG、PNG、GIF、WebP 等）。

### Q: 照片有大小限制嗎？
A: LocalStorage 有 5-10MB 限制，建議單張照片不超過 1MB。

### Q: 可以編輯已上傳的商品�as?
A: 目前版本沒有編輯功能。可以重新上傳。

### Q: 聊天訊息會消失嗎？
A: 訊息儲存在 LocalStorage，除非清除瀏覽器資料，否則會永久保留。

### Q: 可以同時與多個賣家聊天嗎？
A: 可以！每個商品的賣家都是獨立的聊天室。

### Q: 重新整理頁面會怎樣？
A: 所有資料都會保留（商品、購物車、聊天記錄）。

## 🎯 使用情境範例

### 情境 1：買家購物流程
1. 開啟網站，瀏覽首頁商品
2. 使用搜尋列找「iPhone」
3. 點擊商品查看詳情
4. 查看照片與描述
5. 點擊「聊聊」詢問賣家問題
6. 滿意後點擊「加入購物車」
7. 繼續逛其他商品
8. 點擊購物車圖標查看
9. 調整數量後點擊「前往結帳」
10. 填寫收件資訊並確認訂單
11. 完成購買！

### 情境 2：賣家上傳商品流程
1. 點擊「上傳商品」按鈕
2. 拖放 3 張商品照片
3. 填寫標題：「iPhone 15 Pro Max 256GB」
4. 填寫詳細描述：狀況、配件、保固等
5. 輸入價格：42900
6. 選擇狀態：全新
7. 選擇分類：3C產品
8. 點擊「上傳商品」
9. 商品成功顯示在首頁！
10. 等待買家聯繫

### 情境 3：買賣雙方溝通流程
1. 買家在商品詳情頁點擊「聊聊」
2. 輸入訊息：「請問還有現貨嗎？」
3. 發送訊息
4. （模擬）賣家回覆：「有的，台灣公司貨」
5. 買家繼續詢問：「可以面交嗎？」
6. 雙方透過聊天室溝通細節
7. 達成交易共識
8. 買家完成購買流程

## 🔧 開發者說明

### 清除所有資料
在瀏覽器 Console 輸入：
```javascript
localStorage.clear();
location.reload();
```

### 查看儲存的資料
```javascript
// 商品
console.log(JSON.parse(localStorage.getItem('products')));

// 購物車
console.log(JSON.parse(localStorage.getItem('cart')));

// 聊天
console.log(JSON.parse(localStorage.getItem('chats')));
```

### 手動新增範例對話
在商品詳情頁點擊「聊聊」按鈕即可建立與該賣家的聊天室。
